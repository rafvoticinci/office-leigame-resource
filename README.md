# office-leigame-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面资源。页面内容涵盖办公辅助、小游戏、工具页面等类型，均为独立可运行的静态页面，不依赖特定后端服务或域名。

仓库以资源归档为主要目的，方便开发者或用户浏览、下载或引用其中的页面。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
├── docs/           # 相关文档或说明（可选）
└── README.md       # 本文件
```

- `pages/` 目录下的每个 HTML 文件均可独立打开运行。
- `assets/` 目录按页面或功能分类存放资源文件，与 `pages/` 中的页面对应。
- `docs/` 目录用于存放附加说明文档，非必须。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开。
- 归档页面可能包含简单的交互逻辑（如 JavaScript 实现的小游戏或工具），但不会收集用户数据或发送网络请求。
- 页面不依赖任何外部 API 或第三方服务，所有资源均内置于本仓库。
- 部分页面可能引用本地资源（如图片、字体），请确保资源路径正确。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修复已有页面的问题。
- 欢迎提交 Issue 反馈页面问题或提出建议。
- 如需引用或分发本仓库中的页面，请遵守开源协议（详见 LICENSE 文件）。
- 不建议直接修改仓库中已归档的页面文件，以免影响其他用户的引用。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在 `pages/` 目录中找到感兴趣的 HTML 文件。
3. 使用浏览器直接打开该文件即可运行。
